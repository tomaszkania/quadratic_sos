# TOMS submission checklist

This repository is prepared for submission as a TOMS Algorithm-paper software component.
The following checklist is intended for the editor and referees.

## Article

- [x] Article PDF included in `paper/`.
- [x] Article contains ACM CCS descriptors.
- [x] Article contains a software availability and reproducibility subsection.
- [x] Article distinguishes deterministic structural outputs from machine-dependent timings.
- [x] Article identifies the public repository URL.

## Software component

- [x] Installable Python package in `src/`.
- [x] No runtime dependencies outside the Python standard library for the core routines.
- [x] MIT licence and citation metadata supplied.
- [x] Non-interactive smoke-test driver: `python scripts/run_all_checks.py`.
- [x] Regression tests and validation scripts supplied.
- [x] Data and figure reproduction scripts supplied.
- [x] Expected-output summary generated by the smoke-test driver.
- [x] Executed notebook supplied for examples.
