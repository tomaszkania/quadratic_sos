# Changelog

## 0.2.0 - TOMS submission refresh

- Converted the manuscript to ACM `acmart` single-column review format for TOMS.
- Added ACM CCS descriptors, keywords, figure descriptions, and BibTeX references.
- Refreshed benchmark tables and regenerated performance figures from the package API.
- Added package-level `__version__` and updated project and citation metadata.
- Tightened the reproducibility documentation and submission checklist.
- Added a non-interactive submission-archive builder.
- Kept the core package dependency-free apart from the Python standard library.
